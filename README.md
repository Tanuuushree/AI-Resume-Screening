# 📄 AI Resume Screening & Job Matching System

An AI-powered ATS Resume Screening & Job Matching System built using Machine Learning, NLP, and Streamlit.

This application helps recruiters and candidates analyze resumes intelligently by comparing resumes with job descriptions, calculating ATS scores, identifying missing skills, and predicting eligibility for job roles.

---

# 🚀 Features

✅ Resume Upload using PDF  
✅ Job Description Matching  
✅ ATS Resume Match Score  
✅ Smart Resume Category Prediction  
✅ Skill Extraction from Resume  
✅ Required Skills Detection  
✅ Recommended Skills to Learn  
✅ Eligibility Prediction  
✅ Recruiter AI Feedback  
✅ Interactive Streamlit UI  

---

# 🧠 How It Works

1. Recruiter enters Job Description / Required Skills
2. Candidate uploads Resume PDF
3. System extracts resume text
4. NLP processes and analyzes content
5. Skills are matched with job requirements
6. ATS Match Score is generated
7. Eligibility and recruiter feedback are displayed
