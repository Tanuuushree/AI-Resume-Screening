# AI Resume Screening System

AI-powered Resume Screening System using Machine Learning and NLP.

## Features
- Resume Category Prediction
- Skill Extraction
- ATS Resume Score
- Skill Match Percentage
- Missing Skills Detection
- PDF Resume Upload

## Technologies Used
- Python
- Machine Learning
- NLP
- Streamlit
- Scikit-learn

## Deployment
Deployed using Streamlit Cloud.
